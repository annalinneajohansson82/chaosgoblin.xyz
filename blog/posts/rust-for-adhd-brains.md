---
title: Why Rust appeals to ADHD developers
date: 2024-10-12
tags:
  - Programming
  - Rust
  - ADHD
---

Rust's compiler doesn't let you get away with vagueness. Everything must be explicit. Ownership must be clear. This feels like a feature, not a bug, when your brain already struggles with implicit assumptions and ambiguity.

<!--more-->

The strict type system and borrow checker feel less like restrictions and more like scaffolding. They force you to think clearly about what you're doing, which is something ADHD brains often need help with.

The trade-off is that Rust's learning curve is steep. But once you're past it, the enforced clarity becomes a superpower. You write less code that works more reliably.
